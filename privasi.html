Pendahuluan

Selamat datang di KHB Fast Downloader Pro. Kami sangat menghargai privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana aplikasi kami menangani informasi Anda saat Anda menggunakan layanan pengunduhan konten media kami.

Informasi yang Kami Kumpulkan

KHB Fast Downloader Pro dirancang dengan prinsip minim data. Kami tidak mengharuskan Anda untuk membuat akun atau memberikan data pribadi seperti nama, nomor telepon, atau alamat rumah.

  • Data Tautan (URL): Kami memproses tautan yang Anda tempelkan untuk diubah menjadi link unduhan.
  • Kunci API (API Key): Kunci Botcahx dan Kunci Premium disimpan secara lokal di perangkat Anda (local storage) dan tidak dikirim ke server kami.
  • Log Teknis: Kami tidak mencatat riwayat unduhan Anda di server pusat kami.

Penggunaan API Pihak Ketiga

Aplikasi kami menggunakan layanan Botcahx API untuk memproses permintaan unduhan. Saat Anda menekan tombol download:

Tautan yang Anda berikan dikirim ke server Botcahx secara terenkripsi untuk mendapatkan link file media asli. Penggunaan data oleh Botcahx tunduk pada kebijakan privasi mereka sendiri. Kami menyarankan Anda untuk meninjau kebijakan mereka secara berkala.

Keamanan Data

Keamanan data Anda adalah prioritas kami. Kami menggunakan protokol HTTPS untuk setiap komunikasi antara aplikasi dan API. Kami tidak memiliki akses ke galeri atau file pribadi di perangkat Anda selain folder tujuan pengunduhan yang telah Anda izinkan.
